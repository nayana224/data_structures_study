# 1.1 자료구조란?
## 복합 자료 구조
- 선형 자료구조
  - 일렬로 나열 가능
  - 반드시 순서 존재 
  - (ex) 리스트, 스택, 큐, 덱
- 비선형 자료구조
  - 한줄로 나열하기 어려움
  - (ex) 트리, 그래프

## 자료구조의 구현 방법
- 배열구조
  - 자료를 배열에 모아 저장
  - 인접한 메모리 공간에 저장
- 연결된 구조
  - 흩여져 있는 자료들을 연결하여 하나로 관리
  
# 1.2 알고리즘이란?
- 알고리즘: 주어진 문제를 해결하기 위한 단계적인 절차
  
- 유사코드(psedu code) 구현
  - 기존 언어들과 유사
  - 대입(할당) 연산자로 '<-'를 사용
  - '='는 비교 연산자로 사용
